## Input 요소

다양한 타입의 input 요소를 사용하여 사용자로부터 입력을 받을 수 있다.

text : 텍스트 입력

password : 비밀번호 입력

radio : 라디오 버튼

checkbox : 체크박스

file : 파일 선택

select : 선택 입력

textarea : 문장 입력

button : 버튼 입력

submit : 전송 버튼

filedset : 필드 셋
